# Take-Note 
A note-taking application built with Flutter. Create, organize, and manage your notes effortlessly with a clean and intuitive user interface.

## Features
-  **Create & Edit Notes** - Write and edit notes with ease
-  **Clean UI** - Modern and intuitive user interface
-  **Cross-Platform** - Available on iOS and Android

## Tech Stack 
- **Framework:** Flutter
- **Language:** Dart
- **Architecture:** Clean Architecture with Provider State Management

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Imtiaz-Fuad/Take-Note.git
   cd Take-Note
   ```

2. **Get dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the app:**
   ```bash
   flutter run
   ```
