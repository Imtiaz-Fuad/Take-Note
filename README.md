# Take-Note 📝

A modern, feature-rich note-taking application built with Flutter. Create, organize, and manage your notes effortlessly with a clean and intuitive user interface.

**Live Demo:** [https://take-note-tau.vercel.app](https://take-note-tau.vercel.app)

## Features ✨

- 📌 **Create & Edit Notes** - Write and edit notes with ease
- 🏷️ **Organize Notes** - Categorize and tag your notes for better organization
- 🔍 **Search Functionality** - Quickly find notes using powerful search
- 💾 **Auto-Save** - Never lose your work with automatic saving
- 🎨 **Clean UI** - Modern and intuitive user interface
- 📱 **Cross-Platform** - Available on iOS and Android

## Tech Stack 🛠️

- **Framework:** Flutter
- **Language:** Dart
- **Architecture:** Clean Architecture with State Management

## Getting Started 🚀

### Prerequisites

Before you begin, ensure you have the following installed:
- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- [Dart SDK](https://dart.dev/get-dart)
- An IDE (Android Studio, VS Code, or Xcode)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Imtiaz-Fuad/Take-Note.git
   cd Take-Note
   ```

2. **Get dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the app:**
   ```bash
   flutter run
   ```

### Build for Release

**iOS:**
```bash
flutter build ios --release
```

**Android:**
```bash
flutter build apk --release
```
